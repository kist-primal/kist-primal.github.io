selected publications

  1. RA-L
    Robust LiDAR-Visual SLAM in Dynamic Environments
    Minjun Kim, Soyeon Park, and Hyunwoo Lee
    IEEE Robotics and Automation Letters (RA-L), 2025
    Abs Bib PDF

    We propose a novel LiDAR-visual SLAM framework that maintains robust performance in highly dynamic environments by leveraging semantic segmentation for dynamic object filtering.

  2. ICRA
    Cooperative Path Planning for Multi-UAV Coverage
    Soyeon Park and Minji Choi
    In IEEE International Conference on Robotics and Automation (ICRA), 2025
    Abs Bib PDF Code

    We present a decentralized cooperative path planning algorithm for multi-UAV coverage tasks that achieves near-optimal area coverage while minimizing communication overhead.

  3. IROS
    Real-time 3D Object Detection on Edge Devices
    Hyunwoo Lee, Minjun Kim, and Yujin Seo
    In IEEE/RSJ International Conference on Intelligent Robots and Systems (IROS), 2024
    Abs Bib PDF

    We introduce a lightweight 3D object detection architecture optimized for deployment on edge computing platforms, achieving real-time inference with minimal accuracy loss.
